# docker-compose で Hubot + Slack 構築

## 実行手順

```
macOS%$ git clone https://github.com/medpeer-inc/hubot-slack-on-docker
macOS%$ cd hubot-slack-on-docker
macOS%$ docker-compose up -d
```
